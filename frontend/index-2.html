Font Family

H1 Header

Page Heading

To be used for the main heading or title of a page.

font-size: 40px;  font-weight: 700;  line-height: 46px;  margin-bottom: 20px;

H2 Header

Page Heading 2/ Modal

Used for the main heading or title of a page when there are 2 or more levels. Used for the main heading of a modal.

font-size: 32px;  font-weight: 700;  line-height: 38px;  margin-bottom: 20px;

H3 Header

Section Title

Used for titles of sections within a page. Examples include "Water System Information" or "Key Personnel" within the main water system page.

font-size: 28px; font-weight: 700; line-height: 34px;  margin-bottom: 20px;

H4 Header

Subsection Title

Used for subsection titles. These are sections like "Main Phone" or "Address" within the "Water System Information" section.

font-size: 24px; font-weight: 700; line-height: 30px;  margin-bottom: 20px;

H5 Header

Subsection Title

Used for subsection titles. These are sections like "Main Phone" or "Address" within the "Water System Information" section.

font-size: 20px; font-weight: 600; line-height: 26px;  margin-bottom: 12px;

Text Large

Public Pages/ Modal

Used for text on modals or body text on public facing pages to increase readability.

.text-lg
font-size: 19px; line-height: 25px; font-weight: 500;

Text Medium

Body/General

For main body text on most pages.

.text-md
font-size: 16px; line-height: 22px; font-weight: 500;

Text Small

Explanatory Text/ Buttons

For text that is explanatory or lower in hierarchy to the main text. For most regular buttons.
